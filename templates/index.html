如何使用？

  1. 登录您的 GitHub 账户
  2. 上传包含 package.json 的 MicroPython 包
  3. 其他用户可以通过 mip 工具安装您的包

package.json 必须包含:

{
  "name": "example-package",
  "urls": [
    ["example.py", "src/example.py"]
  ],
  "version": "1.0.0"
}

package.json 可选字段:

{
  "_comments": {
    "chips": "该包支持运行的芯片型号，缺少表示无芯片限制",
    "fw": "该包依赖的特定固件如ulab、lvgl,缺少表示无固件依赖"
  },
  "description": "这是一个示例包",
  "author": "Your Name",
  "license": "MIT",
  "chips": ["STM32F407", "ESP32"],
  "fw": ["ulab", "lvgl"]
}
