如何使用？

  1. 登录您的 GitHub 账户
  2. 上传包含 package.json 的 MicroPython 包
  3. package.json 必须包含 name,version,urls字段
  4. 其他用户可以通过 mip 工具安装您的包
示例 package.json:
{
  "name": "example-package",
  "urls": [
    ["example.py", "src/example.py"]
  ],
  "version": "1.0.0",
  "description": "这是一个示例包",
  "author": "Your Name",
  "license": "MIT"
}
