package.json 示例 

{
  "name": "my-micropython-package",
  "version": "1.0.0",
  "urls": [
    ["package.py", "src/package.py"]
  ],
  "description": "我的 MicroPython 包描述",
  "author": "Your Name",
  "license": "MIT",
  "keywords": ["micropython", "iot", "sensor"],
  "url": "https://github.com/username/repository",
  "dependencies": {
    "other-package": ">=1.0.0"
  }
}

可选字段说明

chips

该包支持运行的芯片型号，缺少表示无芯片限制

fw

该包依赖的特定固件如ulab、lvgl，缺少表示无固件依赖

dependencies

包的依赖关系，使用语义化版本号
